# Advanced_Keylogger
# AIM  
To track kids activity including keystrokes, audio, screenshots, clipboard data and System information

# Components Required
1. Pycharm IDE
2. Downloading necessary packages/libraries:
   * email
   * smtplib
   * socket
   * platform
   * win32clipboard
   * pynput
   * time
   * os
   * scipy
   * sounddevice
   * cryptography
   * requests
   * PIL
3. System with minimum 4GB RAM
